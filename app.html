🎨 AI Graphic Designer Studio

An AI-powered design assistant for creating stunning visuals

✨ Key Features

  • 🤖 AI-Powered Design
  • 🎨 Brand-Aware Creation
  • 🌈 Dynamic Color Palettes
  • 🖼️ Multiple Formats
  • 💾 Save & Load Sessions

🚀 How It Works

  1. 1. Upload your logo and brand assets
  2. 2. Choose a design format
  3. 3. Chat with AI to create designs
  4. 4. Iterate and refine
  5. 5. Download your creation

🛠️ Tech Stack

  • React + TypeScript
  • Tailwind CSS
  • Google Gemini API
  • Client-side only

🏁 Getting Started

This is a client-side application that requires a Google Gemini API key.

To run locally:

npm install -g serve && serve .
